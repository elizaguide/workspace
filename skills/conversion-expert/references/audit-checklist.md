# Conversion Audit Checklist

Post-build checklist to verify conversion optimization. Run after page is complete.

---

## Quick Audit (5 items)

Run this for every page:

- [ ] **Value prop visible** — Can visitor understand the offer in 3 seconds?
- [ ] **CTA above fold** — Is primary CTA visible without scrolling?
- [ ] **Social proof present** — Is there a trust indicator (numbers, logos, testimonials)?
- [ ] **Mobile functional** — Does page work on phone? Is CTA tappable?
- [ ] **Guarantee visible** — Is risk reversal mentioned near pricing/CTA?

---

## Priority Fixes

If score is below 45, prioritize fixes in this order:

1. **Hero section** — First impression, highest impact
2. **CTA visibility** — Can't convert if CTA is hidden
3. **Social proof** — Trust drives action
4. **Mobile experience** — 70%+ of traffic
5. **Objection handling** — Remove purchase barriers
6. **Page speed** — Slow pages lose visitors

## Pre-Launch Checklist

Before going live:

- [ ] Spell-check all copy
- [ ] Test all links/buttons
- [ ] Verify checkout URL works
- [ ] Test on iPhone Safari
- [ ] Test on Android Chrome
- [ ] Check Core Web Vitals (PageSpeed Insights)
- [ ] Verify tracking pixels (if applicable)
- [ ] Preview OG tags (Facebook, LinkedIn)
