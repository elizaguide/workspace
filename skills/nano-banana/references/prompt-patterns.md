# Prompt Patterns for Image Generation

Effective prompts for generating high-quality images with Nano Banana Pro.

## Prompt Structure

```
[Subject] + [Style/Medium] + [Composition] + [Mood/Atmosphere] + [Technical Details]
```

## Web Design Patterns

### Hero Backgrounds

```
Abstract gradient background, [color palette], flowing organic shapes,
[mood] atmosphere, high resolution, suitable for website hero section
```

**Examples:**
```
"Abstract gradient with purple and soft pink tones, flowing organic curves,
calm serene atmosphere, suitable for wellness website hero, high resolution"

"Dynamic geometric pattern, deep purple (#7a12d4) and white,
modern minimal design, professional tech company hero background"

"Soft watercolor texture, lavender and cream tones, subtle brush strokes,
elegant feminine aesthetic, spa website hero background"
```

### Feature/Card Images

```
[Subject] [action/state], [style] style, [color accents],
clean modern design, suitable for feature card
```

**Examples:**
```
"Person meditating peacefully, minimal illustration style, purple accents,
soft natural lighting, wellness feature image"

"Stack of books with glowing lightbulb, flat design illustration,
vibrant purple and gold, education feature image"

"Abstract brain visualization with neural connections, 3D render style,
purple and teal gradients, technology feature image"
```

### Icons

```
Minimal [icon type] icon, flat design, [color], simple geometric shapes,
clean lines, suitable for web UI
```

**Examples:**
```
"Minimal heart icon, flat design, purple #7a12d4, simple rounded shape,
clean lines, suitable for web UI"

"Lotus flower icon, minimal flat style, purple gradient,
symmetrical design, meditation app icon"

"Lightning bolt icon, bold flat design, vibrant purple,
sharp geometric angles, energy/power concept"
```

### Avatars/Portraits

```
[Person type] portrait, [setting], [lighting],
professional quality, suitable for profile picture
```

**Examples:**
```
"Professional woman portrait, neutral studio background,
soft natural lighting, warm smile, corporate headshot style"

"Meditation teacher portrait, serene expression, soft focus background,
peaceful natural lighting, wellness brand aesthetic"

"Abstract avatar placeholder, geometric face silhouette, purple tones,
modern minimal design, gender-neutral"
```

### Product/App Screenshots

```
[Device type] showing [app/content], [environment],
[lighting], product photography style
```

**Examples:**
```
"Smartphone displaying meditation app, minimalist desk setup,
soft morning light, lifestyle product photography"

"Laptop screen showing dashboard interface, modern home office,
natural window lighting, tech product shot"
```

## Style Modifiers

### Photography Styles
- `professional photography` - Clean, well-lit
- `lifestyle photography` - Natural, candid feel
- `studio photography` - Controlled lighting, neutral backgrounds
- `editorial photography` - Magazine-quality, artistic

### Illustration Styles
- `flat design illustration` - 2D, minimal shadows
- `vector illustration` - Clean lines, scalable look
- `watercolor illustration` - Soft, organic feel
- `line art illustration` - Minimal, elegant

### 3D Styles
- `3D render` - Realistic materials
- `soft 3D` - Rounded, friendly
- `isometric 3D` - Technical, precise
- `clay render` - Matte, sculptural

### Abstract Styles
- `abstract gradient` - Color transitions
- `geometric abstract` - Shapes and patterns
- `organic abstract` - Natural flowing forms
- `minimalist abstract` - Simple, clean

## Composition Guidelines

### Centered Subject
```
"centered composition, subject in middle of frame"
```

### Negative Space
```
"plenty of negative space, clean minimal composition"
```

### Rule of Thirds
```
"subject positioned using rule of thirds"
```

### Full Bleed
```
"edge-to-edge design, no borders"
```

## Quality Enhancers

Add these for better results:
- `high resolution`
- `professional quality`
- `clean design`
- `well-balanced composition`
- `harmonious colors`
- `clear focal point`

## Negative Prompts (What to Avoid)

When quality issues occur, try excluding:
- "no text" - Prevents unwanted words
- "simple background" - Avoids busy backgrounds
- "clean composition" - Reduces clutter
- "single subject" - Prevents multiple subjects

## Common Issues and Fixes

| Issue | Fix |
|-------|-----|
| Too busy | Add "minimal", "clean", "simple" |
| Wrong colors | Specify hex codes or color names explicitly |
| Bad composition | Add composition guidance ("centered", "negative space") |
| Wrong style | Be more explicit about medium ("flat illustration", "photo") |
| Low quality feel | Add "professional quality", "high resolution" |
